Set A - 5 Points Each Question

  1. How to create reactive forms?

    Answer: To create reactive forms in Angular, import ReactiveFormsModule in your module, define a FormGroup in your component, and bind it to the template using the [formGroup] directive.

  2. How to add built-in validation for email in reactive form?

    Answer: Use the Validators.email validator in the FormControl definition.
    Example:
    email: new FormControl('', [Validators.required, Validators.email])

  3. How to add built-in validation for required fields in reactive form?

    Answer: Use the Validators.required validator in the FormControl definition.
    Example:
    name: new FormControl('', Validators.required)

  4. How to add regex validation for reactive forms?

    Answer: Use Validators.pattern with the desired regex pattern in the FormControl.
    Example:
    mobile: new FormControl('', Validators.pattern('^[0-9]{10}$'))

  5. How to generate routing?

    Answer: Use the Angular CLI command: ng generate module app-routing --flat --module=app

  6. How to configure routes with parameters?

    Answer: Define routes with parameters in the Routes array using the path property.
    Example:
    { path: 'user/:id', component: UserComponent }

  7. What is DebounceTime in RxJS?

    Answer: debounceTime is an RxJS operator that delays the emission of values from an observable by a specified time. It is commonly used to reduce the frequency of events like user input.

  8. What is pipe operator?

    Answer: The pipe operator in RxJS is used to chain multiple operators together to transform or filter observable data.

  9. What is distinctUntilChanged operator?

    Answer: distinctUntilChanged is an RxJS operator that prevents the emission of duplicate consecutive values from an observable.

  10. What is filter operator in RxJS?

    Answer: The filter operator is used to emit values from an observable that satisfy a specified condition.
